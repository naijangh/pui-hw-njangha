Instructions

  1. From the table of ingredients, select the ones you think go into the beverage one at a time. The ingredient will turn into a green check if it is correct or a red x if it is incorrect.
  2. If you need a hint, click on the "Hint" button to learn how many ingredients there go into the drink
  3. Click the "Submit" button if you think you are done. The cup will fill up and give you a rating (based on how many you guessed right or wrong) if all the ingredients are guessed.
  4. If you are missing ingredients and click "Submit", you will encounter a pop-up that alerts you that you need to add more ingredients.
  5. Click the "Back" button to return to the other drink options to play again.
